  1. What are the advantages of Bootstrap?

    2. The Benefits of Using Bootstrap Framework

    • Easy to prevent repetitions among multiple projects.
    • Responsive design that can be used to adapt screen sizes and choose what shows and what doesn't on any given device.
    • Maintaining consistency among projects when using multiple developer teams.
    • Quick design of prototypes.

  2. What is a Bootstrap Container, and how does it work?

    3. Containers are the most basic layout element in Bootstrap and are required when using our default grid system. Containers are used to contain, pad, and (sometimes) center the content within them. While containers can be nested, most layouts do not require a nested container.

  3. What are the default Bootstrap text settings?

    4. Bootstrap uses a default font-size of 1rem (16px by default), and its line-height is 1.5. In addition, all elements have margin-top: 0 and margin-bottom: 1rem (16px by default).

  4. What do you know about the Bootstrap Grid System?

    5. Bootstrap's grid system uses a series of containers, rows, and columns to layout and align content. It's built with flexbox and is fully responsive.

  7. How can you use Bootstrap to make thumbnails?

    8. Bootstrap - Thumbnails

    1. Add an >a< tag with the class of . thumbnail around an image.
    2. This adds four pixels of padding and a gray border.
